Article Rewriter WordPress Plugin

Overview

Article Rewriter is a powerful WordPress plugin designed to streamline content creation by automatically rewriting articles using advanced cloud-based APIs. With support for multiple AI services and flexible rewriting options, this plugin simplifies the process of generating unique content.

Features

Versatile API Integration

  • Supports multiple AI writing services including:
    • OpenAI
    • DeepSeek
    • Anthropic (Coming Soon)
    • Google Gemini (Coming Soon)

Flexible Rewriting Styles

Choose from multiple writing modes:

  • Standard
  • Formal
  • Casual
  • Creative

Seamless WordPress Integration

  • Compatible with both Gutenberg Block Editor and Classic Editor
  • Intuitive user interface for easy article rewriting

System Requirements

  • WordPress 5.0+
  • PHP 7.2+
  • API keys for desired content rewriting services

Installation Guide

  1. Upload the `article-rewriter` folder to `/wp-content/plugins/`
  2. Activate the plugin through WordPress admin
  3. Navigate to 'Article Rewriter' in the admin menu
  4. Enter your Envato purchase code
  5. Configure API settings

Usage Instructions

Single Article Rewriting

Gutenberg (Block) Editor

  1. Open desired post/page
  2. Click "Rewrite Article" toolbar button
  3. Select preferred API and style
  4. Execute rewrite

Classic Editor

  1. Open desired post/page
  2. Click "Rewrite Article" toolbar button
  3. Select preferred API and style
  4. Execute rewrite

Batch Processing

  1. Go to 'Article Rewriter > Batch' in admin menu
  2. Configure processing parameters:
    • Select post type
    • Choose API
    • Select rewriting style
  3. Set number of posts to process
  4. Initiate batch processing
  5. Monitor progress on batch page

API Configuration

OpenAI Setup

  1. Create account at OpenAI Platform
  2. Generate API key
  3. Enter key in plugin settings

DeepSeek Setup

  1. Create account at DeepSeek Platform
  2. Generate API key
  3. Enter key in plugin settings

Upcoming Integrations

  • Anthropic Claude API: Coming soon
  • Google Gemini API: Coming soon

Licensing

  1. Purchase plugin from Envato
  2. Navigate to 'Article Rewriter > License'
  3. Input Envato purchase code
  4. Activate license

Important: One purchase code is valid for a single domain.
